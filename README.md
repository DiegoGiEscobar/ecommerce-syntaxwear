# 👟 SyntaxWear - E-commerce de Sneakers

Bem-vindo ao repositório do **SyntaxWear**, um projeto de e-commerce moderno focado na venda de tênis e sneakers exclusivos. Este projeto foi desenvolvido para colocar em prática conceitos fundamentais de desenvolvimento web front-end.

---

## 🚀 Sobre o Projeto

A **SyntaxWear** é uma interface de loja online projetada para oferecer uma experiência de usuário fluida e visualmente atraente. O site apresenta uma coleção curada de calçados divididos em categorias como Casual, Esporte, Moderno e Futurista.

### 🎯 Objetivos de Aprendizado
Este projeto foi construído com o foco em:
- Estruturação semântica com **HTML5**.
- Estilização modular e organizada com **CSS3**.
- Design responsivo para dispositivos móveis e desktop.
- Organização de arquivos e boas práticas de arquitetura CSS.

---

## 🛠️ Tecnologias Utilizadas

- **HTML5**: Estrutura robusta e acessível.
- **CSS3**: Estilização moderna utilizando:
    - **Variáveis CSS**: Para facilitar a manutenção de cores e fontes.
    - **Flexbox e CSS Grid**: Para layouts complexos e responsivos.
    - **Metodologia de Componentes**: Estilos divididos em arquivos menores para melhor organização.

---

## 📂 Estrutura de Pastas

A organização do projeto segue uma estrutura profissional:

```text
ecommerce-syntaxwear/
├── css/
│   ├── components/      # Estilos específicos de cada parte do site (header, footer, etc)
│   ├── base.css         # Estilos base e globais
│   ├── reset.css        # Limpeza de estilos padrão do navegador
│   ├── variables.css    # Definição de cores, fontes e espaçamentos
│   └── layout.css       # Estrutura principal da página
├── images/              # Ativos visuais (logos, ícones e fotos de produtos)
├── index.html           # Página principal do site
└── README.md            # Documentação do projeto
```

## 🚀 Como Visualizar o Projeto

Para rodar este projeto localmente, você não precisa de nenhuma instalação complexa:

1. Faça o download ou clone este repositório:
   ```bash
   git clone https://github.com/DiegoGiEscobar/ecommerce-syntaxwear.git
   ```
2. Navegue até a pasta do projeto.
3. Abra o arquivo `index.html` em qualquer navegador (Chrome, Firefox, Edge).

*Dica: Se você usa o VS Code, recomendo a extensão **Live Server** para visualizar as alterações em tempo real.*
